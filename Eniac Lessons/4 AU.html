Html
Linguagem de Marcação de Hipertexto. Usada para estruturar o conteúdo de uma página web.
Css
Folha de Estilo em Cascata. Usada para estilizar a aparência de documentos Html.
Javascript
Linguagem de programação usada para criar interatividade em páginas web.
